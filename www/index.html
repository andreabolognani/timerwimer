Help

Back

Timer Wimer allows you to manage multiple countdown timers. Each timer is assigned a target time and, optionally, a label.

To create a new timer, open the application menu and select Add.

Once a timer has been created, tap on it to start the countdown. Tapping a timer while it’s running will pause it, and tapping it a second time will resume it. When the countdown ends, a sound will be played.

You can long–press a timer on the main screen to perform other operations on it: you can Reset a running timer to restore it to its initial state; you can Edit a timer to change its target time or label; you can Delete a timer you no longer need (be careful, as this cannot be undone).

Warning!

The timers used in this application have been designed to provide sub–second accuracy, but there’s no guarantee that they actually do. If you find yourself in a situation where having an accurate timer is paramount, you might want to look somewhere else. Your tea will be fine, though.

It’s recommended to keep the application in the foreground while timers are running: failing to do so might prevent the countdown from ever ending.
